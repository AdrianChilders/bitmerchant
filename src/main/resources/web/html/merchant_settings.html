Warning: Server not secure!

To enable SSL/TLS encryption(which you definitely should since as a merchant you will be sending and receiving money), You need to place two files in ~/.bitmerchant-wallet/, and then restart bitmerchant. This warning will dissapear once this is done.

  • A .jks java keystore file named keystore.jks
    • You can use these keytool commands to generate one:
      openssl pkcs12 -export -in public.crt -inkey privatekey.key -out server.p12 -name pkcs12alias -CAfile intermediate.crt -caname root

      keytool -importkeystore -deststorepass farbar -destkeypass farbar -destkeystore keystore.jks -srckeystore server.p12 -srcstoretype PKCS12 -srcstorepass farbar -alias pkcs12alias
  • a file named pass, containing your keystore password.
